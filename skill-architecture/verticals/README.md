## 统一蓝图模板

每个行业蓝图包含 7 个标准章节：

1. **行业特征与约束**：识别行业特性对Skill设计的影响
2. **核心场景地图**：10+ 个高价值 L3 场景及其L2复用
3. **L0/L1 行业专用Skill**：连接器矩阵 + 行业专属基础能力
4. **标杆场景详设**：1 个端到端 TaskFlow YAML
5. **多Agent协作场景**：3+ 个 L4 协作模式
6. **治理红线**：行业合规与伦理底线
7. **实施路线**：3-4 个 Phase 的渐进路径

## 蓝图编写原则

- **不重复造轮子**：所有 L3 必须基于已设计的 L2 模式（router/document/alert/taskflow）
- **行业治理优先**：先识别合规红线，再设计能力
- **标杆场景可演示**：每个行业选 1 个最有代表性的场景做端到端 YAML
- **L4 是远期目标**：多Agent是行业成熟后的形态，先打牢 L0-L3
